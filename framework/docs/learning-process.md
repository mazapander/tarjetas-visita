# Learning Process

This framework is the result of learning by building.

1. Built small things manually: React apps, APIs, scrapers, dashboards and Docker stacks.
2. Repeated the same foundations too many times: auth, users, admin, database, envs and deployment.
3. Adopted AI-assisted development, then discovered new problems: assumptions, overengineering and large diffs.
4. Moved from asking for code to designing issues.
5. Added repo-level instructions: AGENTS.md, CODEX.md, Cursor rules and compatible agent docs.
6. Optimized context with Repomix, Gitingest and file-level prompts.
7. Productized the workflow into this reusable framework.
