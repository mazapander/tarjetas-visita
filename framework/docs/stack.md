# Stack

## Backend
Python, FastAPI, SQLAlchemy 2.x, Alembic, PostgreSQL, Pydantic Settings, Authlib/PyJWT, Uvicorn, Pytest.

## Frontend
React, Vite, TypeScript, React Router, TanStack Query, Tailwind CSS, shadcn/ui, React Hook Form, Zod.

## Auth
OIDC-first: Azure Entra ID, Google, Keycloak and external providers.

## Infra
Docker Compose, VPS, reverse proxy, `.env.example`, no committed secrets.

## AI
Codex, Cursor, ChatGPT, Claude Code-compatible instructions, Repomix, Gitingest, GitHub Issues/PRs.
